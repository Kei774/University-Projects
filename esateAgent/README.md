# EsateAgent

A frontend web application built using React and Vite.

This project helped me understand modern frontend development, component structure, and clean project organization.

## Overview

EsateAgent is structured as a scalable React application with reusable components and modular folders.

Tech stack:

* React
* Vite
* JavaScript
* CSS
* ESLint
* Jest

## Project Structure

src

* components
* pages
* assets
* data
* **mocks**

Main files:

* App.jsx
* main.jsx
* vite.config.js
* package.json

## What I Learned

Component Based Architecture
I learned how to break UI into reusable components and organize them properly.

State and Props
I practiced passing data between components and managing UI state.

Project Structure
I structured the project into pages, components, and assets to keep it scalable and clean.

Vite Setup
I understood how Vite works as a fast development server and bundler.

Testing Basics
I explored Jest and mock folders for testing components.

Linting and Code Quality
I used ESLint to maintain consistent code standards.

## How to Run

1. Install dependencies

npm install

2. Start development server

npm run dev

3. Build for production

npm run build

## Why I Built This

I built this project to strengthen my frontend fundamentals and understand how real world React applications are structured and maintained.

This project improved my understanding of clean architecture, reusable components, and modern tooling.
