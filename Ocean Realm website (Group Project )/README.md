# Ocean Realm Website

A multi page static website built as a group project.

This project helped me understand structured web development using HTML and CSS, along with teamwork and validation practices.

## Overview

Ocean Realm is a themed website that includes:

* Home page
* Content pages
* Profile page
* Feedback form
* Sitemap
* Team page
* Volunteer page
* Table page
* Splash screen

The project follows a structured page layout with consistent styling and navigation across all pages.

Tech used:

* HTML5
* CSS3

## Project Structure

Folders:

* assets
* css
* validationSnapshots

Main pages:

* home.html
* splash.html
* sitemap.html
* team.html
* profile.html
* feedback.html
* volunteer.html
* table.html
* content_ST1 to content_ST4
* pageEditor_ST1 to pageEditor_ST4
* validation_ST1 to validation_ST4

## What I Learned

Semantic HTML
I used proper tags to structure content clearly and improve readability.

Consistent Layout Design
I built reusable navigation and layout patterns across multiple pages.

Form Handling
I created structured forms with validation for the feedback page.

Collaboration
I worked in a team environment, divided responsibilities, and maintained consistent design standards.

Validation
I validated pages using HTML and CSS validators and documented results in validationSnapshots.

Project Organization
I maintained a clean folder structure for scalability and clarity.

## How to Run

Open home.html in a browser.

No server setup required.

## Why I Built This

This project strengthened my foundation in core web development and taught me how to manage a structured, multi page website in a team environment.

It improved my understanding of clean markup, consistent styling, and collaborative development workflows.
